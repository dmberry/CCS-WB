Frame-Splitting Mechanism

EDUCATIONAL SAMPLE - This demonstrates the technical mechanism used in Olia Lialina's "My Boyfriend Came Back from the War" (1996)

How Frame-Splitting Works

The artwork uses HTML framesets to recursively subdivide the browser viewport. Each click on a hyperlink triggers a new frameset that splits the current frame into smaller sections.

Initial State (Level 0)

<frameset cols="50%,50%">
  <frame src="left.html">
  <frame src="right.html">
</frameset>

This creates a vertical split: two equal-width frames side by side.

After First Click (Level 1)

When user clicks a link in left.html:

<frameset rows="50%,50%">
  <frame src="top.html">
  <frame src="bottom.html">
</frameset>

The left frame subdivides horizontally into top and bottom sections.

Recursive Pattern

Each subsequent frame can be further subdivided:

  • Vertical splits: cols="50%,50%" or other proportions
  • Horizontal splits: rows="50%,50%" or other proportions
  • Multiple splits: cols="33%,33%,34%" for three columns

Frame Accumulation

Unlike page-based hypertext (which replaces content), frame-splitting accumulates content. Previous frames remain visible as new ones appear, creating a palimpsest of text and images.

This accumulation creates visual metaphor for:

  • Fragmented memory
  • Interrupted communication
  • Temporal layering
  • Psychological rupture

Technical Constraints of 1996

  • Bandwidth: 28.8k modems (3.6 KB/sec) - images needed to be tiny
  • Screen resolution: 640x480 or 800x600 typical - frames got very small
  • Browser compatibility: Netscape Navigator 2.0, Internet Explorer 3.0 - limited CSS
  • No JavaScript libraries: Hand-coded HTML only
  • GIF images: Limited to 256 colors, file size restrictions

Why Frames Were Perfect

HTML frames (now deprecated) were ideal for this artwork because:

  1. Native browser feature - no plugins required
  2. Persistent visibility - multiple frames visible simultaneously
  3. Recursive structure - framesets within framesets within framesets
  4. Spatial metaphor - browser window as compositional space
  5. Vernacular aesthetic - frames were amateur/DIY, not professional

Frame-Splitting as Narrative Syntax

The frame structure IS the narrative. Each split decision creates meaning:

  • Direction: Horizontal vs vertical splits have different connotations
  • Proportion: 50/50 equal vs asymmetric divisions
  • Depth: How many levels of nesting?
  • Density: How many frames accumulate?
  • Navigation: Which frames contain clickable links?

This is computational narrative - the code structure IS the storytelling mechanism.
