# Available Skins

Custom visual themes for the CCS Workbench. Enable skins in Settings > Appearance.

## Adding Your Own

1. Create a folder in `public/skins/` with your skin files
2. Add an entry below in the format: `- folder-name: Display Name`
3. See README.md for full documentation
